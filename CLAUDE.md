## Agent skills

Local markdown files — final structured issues in `docs/issues/`, temporary planning notes in `.scratch/`. See `docs/agents/issue-tracker.md`.

使用本地 Markdown 文件管理 issue。`docs/issues/` 存放最终的结构化 issue，`.scratch/` 存放临时的规划和工作笔记。详见 `docs/agents/issue-tracker.md`。

### Triage labels

Local markdown frontmatter labels: needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix. See `docs/agents/triage-labels.md`.

本地 Markdown 的 frontmatter 标签体系。详见 `docs/agents/triage-labels.md`。

### Domain docs

单上下文布局。`CONTEXT.md` 位于仓库根目录，`docs/adr/` 存放架构决策记录。详见 `docs/agents/domain.md`。
